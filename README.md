# jolene.portfolio

**🔗 Live site: https://jolene-portfolio.vercel.app**

> Note: the live URL uses a **hyphen** (`-`), not the dot in the repo name. Typing the dotted form (`jolene.portfolio...`) will not load.

Personal developer portfolio. Built with React, Vite, and Tailwind CSS; deployed on Vercel.

## Run locally

```sh
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
